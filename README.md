AgroSmart

## AgroSmart :

AgroSmart is a data-driven agriculture platform that integrates soil parameters,
geographic location, and real-time weather data to provide intelligent crop
recommendations and yield estimation.

## Features :
- User authentication (signup/login)
- Real-time weather data integration
- Crop recommendation based on soil & climate
- Yield estimation support
- Simple and farmer-friendly UI

## Technologies Used :
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js
- APIs: Weather API
- Database: MongoDB

## How to Run :
1. Clone the repository
2. Install dependencies using `npm install`
3. Run the server using `node server.js`
4. Open the application in your browser
